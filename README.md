
Phase 3 Blog

#Description

The Phase 3 Blog is a simple static website built as part of a coding bootcamp project. It features a basic HTML homepage that links to blog posts written in Markdown format. The content is hosted on GitHub and can be viewed locally or deployed as a static site, demonstrating fundamental web development skills without any backend or database.

#Author

Collins Thuo

#Setup Instructions

Clone this repository
Navigate into project directory
Open index.html in your web browser (no additional tools or environments required)

#BDD (Behavior Driven Development)
Input: Open index.html in browser
Output: Homepage loads with links to blog posts
Input: Click on a blog post link
Output: Blog post content displays in Markdown format (rendered by browser or viewer)
Input: Add a new Markdown file for a post
Output: Update index.html to include a link to the new post
Input: Invalid file path or missing file
Output: Browser shows standard 404 or file not found error
Input: View project on GitHub
Output: Repository displays files and commit history
Input: Exit browser
Output: Session closes successfully
